spark.streaming.dstream.NetworkReceiver

BlockGenerator

class BlockGenerator extends Serializable with Logging

Batches objects created by a spark.streaming.NetworkReceiver and puts them into appropriately named blocks at regular intervals. This class starts two threads, one to periodically start a new batch and prepare the previous batch of as a block, the other to push the blocks into the block manager.
