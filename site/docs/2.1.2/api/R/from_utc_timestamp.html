from_utc_timestamp

Description

Given a timestamp, which corresponds to a certain time of day in UTC, returns another timestamp that corresponds to the same time of day in the given timezone.

Usage

from_utc_timestamp(y, x)

## S4 method for signature 'Column,character'
from_utc_timestamp(y, x)

Arguments

y

Column to compute on.

x

time zone to use.

Note

from_utc_timestamp since 1.5.0
