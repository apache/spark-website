lit

Description

A new Column is created to represent the literal value. If the parameter is a Column, it is returned unchanged.

Usage

lit(x)

## S4 method for signature 'ANY'
lit(x)

Arguments

x

a literal value or a Column.

Note

lit since 1.5.0
