isActive

Description

Returns TRUE if this query is actively running.

Usage

isActive(x)

## S4 method for signature 'StreamingQuery'
isActive(x)

Arguments

x

a StreamingQuery.

Value

TRUE if query is actively running, FALSE if stopped.
