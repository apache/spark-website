sha2

Description

Calculates the SHA-2 family of hash functions of a binary column and returns the value as a hex string.

Usage

## S4 method for signature 'Column,numeric'
sha2(y, x)

sha2(y, x)

Arguments

y

column to compute SHA-2 on.

x

one of 224, 256, 384, or 512.

Note

sha2 since 1.5.0
