SQL Query

Description

Executes a SQL query using Spark, returning the result as a SparkDataFrame.

Usage

## Default S3 method:
sql(sqlQuery)

Arguments

sqlQuery

A character vector containing the SQL query

Value

SparkDataFrame

Note

sql since 1.4.0
