setCheckpointDir {SparkR}R Documentation

Set checkpoint directory

Description

Set the directory under which SparkDataFrame are going to be checkpointed. The directory must be a HDFS path if running on a cluster.

Usage

setCheckpointDir(directory)

Arguments

directory

Directory path to checkpoint to

Note

setCheckpointDir since 2.2.0
