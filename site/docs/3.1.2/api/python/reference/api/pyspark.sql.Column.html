pyspark.sql.Column¶

class pyspark.sql.Column(jc)[source]¶

A column in a DataFrame.

Column instances can be created by:

# 1. Select a column out of a DataFrame

df.colName
df["colName"]

# 2. Create from an expression
df.colName + 1
1 / df.colName

New in version 1.3.0.
