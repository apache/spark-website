showDF

Description

Print the first numRows rows of a DataFrame

Usage

## S4 method for signature 'DataFrame'
showDF(x, numRows = 20, truncate = TRUE)

showDF(x, ...)

Arguments

x

A SparkSQL DataFrame

numRows

The number of rows to print. Defaults to 20.
