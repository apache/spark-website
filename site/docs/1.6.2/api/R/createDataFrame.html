createDataFrame {SparkR}R Documentation

Create a DataFrame

Description

Converts R data.frame or list into DataFrame.

Usage

createDataFrame(sqlContext, data, schema = NULL, samplingRatio = 1)

as.DataFrame(sqlContext, data, schema = NULL, samplingRatio = 1)

Arguments

sqlContext

A SQLContext

data

An RDD or list or data.frame

schema

a list of column names or named list (StructType), optional

Value

an DataFrame

Examples
