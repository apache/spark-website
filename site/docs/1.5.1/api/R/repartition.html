Repartition

Description

Return a new DataFrame that has exactly numPartitions partitions.

Usage

## S4 method for signature 'DataFrame,numeric'
repartition(x, numPartitions)

Arguments

x

A SparkSQL DataFrame

numPartitions

The number of partitions to use.
